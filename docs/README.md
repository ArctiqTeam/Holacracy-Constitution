## How
- __macOS:__

#### Install docker:

`https://docs.docker.com/docker-for-mac/install/#download-docker-for-mac`

#### Clone the repo:

`git clone -b <branch_name> https://github.com/ArctiqTeam/ArctiqTeam.github.io.git`

#### Run docker-compose

`$ cd ArctiqTeam.github.io` <br>
`$ docker-compose up`
